Image Recognition Lead Detection Program

In this project, I solved a tough time for the business. I noticed that many leads were coming when no one would be available to answer. That was causing us to lose the chance to get those crucial clients looking for services we had already filtered in the Nearby jobs settings.

At the time, my only programing language was Python, and with some knowledge from previous “for fun projects,” I knew how to work with the PyautoGUI library. To create this project, I had to learn image recognition; I set up a constant loop to check the screen for the "See Job" button. If found, I would run a macro that would perform clicks on the screen to send a message to the client. Even if my team took hours to respond to the client, we would still be able to get the job since as long as we send an initial message, we are counted in as one of the six exclusive companies allowed to message the client. To be one of the six, you only need to message the client first.

Since it was created, this program has assured us that we would get the lead and close a job with the client. I can ensure a minimum of 40% of our clients today were initiated by this program.

Created: Feb -2023

Image Recognition Lead Detection Program

The Functional Electrical Stimulation is a machine that we use daily at DRIVEN NeuroRecovery Center. The machine controls electrical stimulations to specific muscles in a timed manner to assist in movements. The specific one that I chose to simulate is for an electric bike; after the client is entirely set up, even those without any leg movement can perform the movement to use a bike, much like those you can find at the gym.

During my internship, I noticed that getting used to the interface, muscles, and movements that they would perform took a while. To assist with that, I came up with the idea of creating an online simulator the company will use to train newly hired employees and interns. This project allowed me to push myself beyond CSS, HTML, and JavaScript, along with a start on Blender to create the animations. Please excuse the old tech looks, I kept the designed as close as possible to the original.

The program can be accessed at: https://xbvkano.github.io/FES/
